Depois de meses prototipando, saiu o MVP do **Capivara** — meu gerenciador financeiro pessoal. A ideia é simples: conectar as contas, categorizar automaticamente, e mostrar onde o dinheiro tá indo.

## Stack

- **Backend**: FastAPI + SQLModel
- **Categorização**: LLM local (via Arachne)
- **Frontend**: React + Tailwind
- **Banco**: SQLite (prototipação)

## Funcionalidades v1

- [x] Importação de extrato CSV
- [x] Categorização automática por IA
- [x] Dashboard mensal
- [x] Alertas de gastos atípicos

A cereja do bolo: a categorização acerta ~85% das transações sem treinamento prévio. Com feedback manual, chega a 95%.

> "Gastar é fácil. Saber onde foi que é difícil."
